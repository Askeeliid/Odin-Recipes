# Odin Recipes

A simple recipe website built as part of The Odin Project's Foundations curriculum.

This project demonstrates core HTML skills, including structuring web pages, utilizing headings, paragraphs, lists, images, and links.

## Live Demo

[View Live Project](https://askeeliid.github.io/odin-recipes/)

## Features

- **Homepage:** A main index page welcoming users and linking to recipes.
- **Recipe Pages:** Individual pages for specific dishes.
- **Ordered Lists:** Used for clear, step-by-step cooking instructions.
- **Unordered Lists:** Used for ingredient lists.
- **Images:** Visual representations of the finished dishes.

## Technologies Used

- HTML5

## Reflection

This was my first project from The Odin Project. I learned how to properly structure an HTML document, work with relative file paths for images and linking internal pages, and organize data using lists.
